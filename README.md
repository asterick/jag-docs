# Atari Jaguar Developer Reference

A cross-linked developer reference for the **Atari Jaguar** — the Tom & Jerry core
system and the CD-ROM subsystem — distilled from the original Atari developer
documentation and the community "Version 10" restorations.

## 📖 Read it online

### **https://asterick.github.io/jag-docs/**

## Repository layout

- [`docs/`](docs/) — the documentation, published with GitHub Pages (start at [`docs/index.md`](docs/index.md))
- [`docs/sources/`](docs/sources/index.md) — the original source PDFs (see the [source index](docs/sources/index.md) for descriptions and notes)
- `CLAUDE.md` — notes for maintaining the reference
